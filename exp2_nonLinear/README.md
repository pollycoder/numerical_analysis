# 数值分析——非线性方程组解法

## 问题描述

试用以下方法求方程:
$$x^3+2x^2+10x-20=0$$

在$x_0=1$附近的根.(准确解为$x^*=1.368808107...$)希望精度达到$10^{-7}$(要注意避免死循环):

(1) $x_{k+1}=\frac{20-2x_k^2-x_k^3}{10}$;

(2) $x_{k+1}=\sqrt[3]{20-10x_k-2x_k^2}$;

(3) 方法(1)的Steffensen加速方法;

(4) 方法(2)的Steffensen加速方法;

(5) Newton法;

## 文件结构

1. 主程序：main.m

求解+绘图（可绘出震荡现象的图）

2. 求解程序

直接迭代法：directIter.m

Steffensen加速法：steffen_acc.m

Newton迭代法：newton.m
